After a successful assessment phase, TypeScript has moved to trial. The 911 team has joined Taycan in adopting it, and we now mandate TypeScript for all new frontend projects. Backend adoption via Node.js services is also underway. The type system has measurably reduced production bugs related to data shape mismatches.
